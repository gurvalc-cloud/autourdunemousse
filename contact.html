Contact

Ce site reste une version statique et sans CMS. Le projet est construit en HTML, CSS et JavaScript pour garder la flexibilité et l'indépendance du CMS.

Si tu veux connecter une adresse e-mail ou un formulaire, on peut ajouter une solution simple en JS ou héberger une endpoint externe.

Pour l'instant, voici quelques informations de base :

Nom du site : Autour d'une Mousse

Projet : version statique du site parchemin

Format : HTML / CSS / JS

Prochaine étape :

  • Ajouter un formulaire de contact statique
  • ou connecter un service de formulaire externe pour envoyer des messages
  • ou déployer le site sur un hébergement simplifié
